# Remember your tasks!

Remember your tasks! es un pequeño proyecto educativo para gestionar las tareas pendientes de manera sencilla y organizada.

## Tecnologías utilizadas

Para este proyecto he utilizado diversas tecnologías como pueden ser:

* HTML (Maquetación de la página)
* CSS (Estilos de la página, filosofía mobile first)
* JavaScript (Vanilla)
* AJV (Librería para validación de objetos JSON)
* Vite (Gestor de proyectos para JavaScript)

## Funcionalidades de la página

La página permite añadir tareas con un título identificativo y una breve descripción que permite tener tus tareas o ideas de manera organizada. Hay validaciones que no permiten continuar si se detectan problemas, como por ejemplo si no introduces ningún dato en el formulario. La aplicación también permite eliminar las tareas de manera individual o colectiva.

## Cómo utilizar la aplicación

Algoritmo de uso de la página web:

1. Escribe un título en el campo correspondiente.
2. Escribe una breve descripción en el campo correspondiente.
3. Pulsa el botón de añadir para agregar la tarea a la lista de tareas.
4. Pulsa eliminar sobre la tarea para eliminarla de la lista.
5. Pulsa limpiar lista para borrar todo el contenido de esta.
